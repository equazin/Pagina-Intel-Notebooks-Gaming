¿Cómo obtener tu token? (1 sola vez)

  1. Entrá a github.com/settings/personal-access-tokens/new (logueada con tu cuenta).
  2. En Token name poné cualquier nombre (ej: "Panel portfolio").
  3. En Repository access → Only select repositories → elegí Pagina-Intel-Notebooks-Gaming.
  4. En Permissions → Repository permissions → Contents → poné Read and write.
  5. Clic en Generate token, copialo y pegalo arriba. ¡Listo!

El token es como una llave: da acceso solo a este repositorio. No lo compartas.
