Cómo obtener el token

  1. Abrí la configuración de tokens de GitHub.
  2. En acceso a repositorios, elegí únicamente sophia-maseda-portfolio.
  3. En permisos del repositorio, asigná Contents: Read and write.
  4. Generá el token y pegalo en el campo de configuración.

El token permite modificar el contenido del portfolio. No lo compartas ni lo uses en otro sitio.
